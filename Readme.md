中山大学抢课助手 V0.3版
=====
作者：陈浩川（厚礼）    <- 软院狗    微博：sysu_厚_礼

协议：GPL v2.0
所有能够访问用户第三方信息并且可能主动上传、共享的应用，都应该是开源的，否则既有木马的可能。

编译说明：
请同CURL 7.38.0一同编译。

使用说明：
解压到任意目录下，安装VC++2013运行库12.0.21005.1。
32位系统安装32位，64位系统装64位和32位。
WINDOWS XP系统无法使用。

登录账号密码与选课系统完全相同，一次可以选多个课。
但每次选课间隔为5秒（照顾渣选课系统）。
选课依赖教学班号，教学班号点开选课系统上的课程名称就能看到了。
一切输入完毕后，挂机运行就可以了，选到之后会告诉你的。我亲测有效。

有BUG请在微博反馈。
